# 👨🏼‍💻 Portfolio ─ React + Tailwind Freelancer Portfolio Theme
Portfolio is a React and Tailwind CSS-based project that helps you create and showcase your own portfolio. This README file provides information on how to get started with your project, how to use it, and essential details for developers who want to contribute.

## 🤖 Technology
- [React](react.dev)
- [Vite](vitejs.dev) 
- [Tailwind](tailwindcss.com)

## Installation
```
git clone https://github.com/kaanayyildiz/portfolio.git

cd portfolio

@Edit the /src/data/config.json file.

npm install

npm run dev
```
